#### JS事件委托

---

- ##### 事件委托的作用

  - 支持为同一个DOM元素注册多个同类型事件
  - 可以将事件分为捕获和冒泡执行

- ##### 注册多个事件

  - 正常注册多个事件，会出现覆盖情况

  - 使用委托机制实现：addEventListener(type,listener,useCapture)

    | 参数       | 作用                                                  |
    | ---------- | ----------------------------------------------------- |
    | type       | 必须，String类型，事件类型                            |
    | listener   | 必须，函数体（js方法）                                |
    | useCapture | 可选，默认false，发生在冒泡阶段，true，发生在捕获阶段 |
